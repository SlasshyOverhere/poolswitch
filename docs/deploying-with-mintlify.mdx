## What is already in this repo

This repository includes a Mintlify docs setup:

- `docs.json` at the repo root
- MDX content in the `docs` folder
- a docs structure ready to sync with GitHub

## Local preview

Run the docs locally from the repo root:

```bash
npx mint dev
```

Mintlify will start a local preview server and watch your MDX files.

## Publish with Mintlify

1. Create a Mintlify account.
2. Create a new docs project.
3. Connect the GitHub repository: `SlasshyOverhere/poolswitch`.
4. Point Mintlify at the repo root so it can find `docs.json`.
5. Let Mintlify sync and build the site.

After the GitHub integration is connected, pushes to your docs content will trigger new builds automatically.

## Custom domain

For a polished production URL, use a docs subdomain such as:

- `docs.poolswitch.dev`
- `docs.yourdomain.com`

In Mintlify:

1. Open your project settings.
2. Add the custom domain.
3. Create the DNS record Mintlify asks for.
4. Wait for verification and SSL issuance.

## Recommended workflow

- keep product docs in this repo
- edit MDX in pull requests
- preview locally with `npx mint dev`
- let Mintlify publish from `main`
