
```bash Node.js
npm install poolswitch-node
```

## Choose your mode

PoolSwitch supports two usage patterns:

- **Embedded client**
  Best for Python, Node.js, and TypeScript apps. No separate process to run.
- **Proxy server**
  Best for multi-language teams or shared internal gateways.

<Tabs>
  <Tab title="Embedded Client (Node.js)">
    This is the recommended path if your app is written in JavaScript or TypeScript.

    ```ts
    import { PoolSwitchClient } from "poolswitch-node";

    const client = new PoolSwitchClient({
      upstreamBaseUrl: "https://api.openai.com",
      keys: [
        { id: "openai-free-1", value: process.env.OPENAI_KEY_1! },
        { id: "openai-free-2", value: process.env.OPENAI_KEY_2! }
      ],
      strategy: "quota_failover"
    });

    const response = await client.post("/v1/chat/completions", {
      json: {
        model: "gpt-4o-mini",
        messages: [{ role: "user", content: "Say hello from PoolSwitch." }]
      }
    });

    console.log(response.choices[0].message.content);
    ```

    The client handles:

    - selecting the next key
    - injecting the auth header
    - retrying safe failures
    - cooling down exhausted keys
    - failing over to another account
  </Tab>
  <Tab title="Embedded Client (Python)">
    This is the recommended path if your app is written in Python.

    ```python
    from poolswitch import PoolSwitchClient

    client = PoolSwitchClient(
        upstream_base_url="https://api.openai.com",
        keys=[
            {"id": "openai-free-1", "value": "sk-123"},
            {"id": "openai-free-2", "value": "sk-456"},
        ],
        strategy="quota_failover",
        retry_attempts=3,
        cooldown_seconds=3600,
    )

    response = client.post(
        "/v1/chat/completions",
        json={
            "model": "gpt-4o-mini",
            "messages": [{"role": "user", "content": "hello"}],
        },
    )

    print(response["choices"][0]["message"]["content"])
    ```

    For async Python applications:

    ```python
    from poolswitch import AsyncPoolSwitchClient

    async with AsyncPoolSwitchClient(
        upstream_base_url="https://api.openai.com",
        keys=["sk-123", "sk-456"],
    ) as client:
        response = await client.get("/v1/models")
        print(response["data"][0]["id"])
    ```

  </Tab>
  <Tab title="Proxy Mode">
    Run PoolSwitch as a local server if you want plain HTTP access from any language.

    **1. Create a config file**

    Example:

    ```yaml poolswitch.yaml
    listen_host: 127.0.0.1
    listen_port: 8080
    upstream_base_url: https://api.openai.com
    auth_header_name: Authorization
    auth_scheme: Bearer
    strategy: quota_failover
    keys:
      - id: openai-primary
        value: sk-123
      - id: openai-secondary
        value: sk-456
    ```

    **2. Start the proxy**

    ```bash
    poolswitch start --config poolswitch.yaml
    ```

    **3. Call the local endpoint**

    ```bash
    curl http://127.0.0.1:8080/health
    ```

    Then send provider-compatible requests through PoolSwitch:

    ```bash
    curl -X POST http://127.0.0.1:8080/v1/chat/completions \
      -H "Content-Type: application/json" \
      -d '{"model":"gpt-4o-mini","messages":[{"role":"user","content":"hello"}]}'
    ```
  </Tab>
</Tabs>
